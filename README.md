## Installation
_(You need to have both git and hyprland installed)_
1. Clone the repo
```bash
git clone https://github.com/Pre1ude0/hyprland-config
```
2. Change to the desired branch
```bash
git checkout <branch>
```
3. Copy the config files to their respective directories
```bash
cp -r <config>/* ~/.config/
```
